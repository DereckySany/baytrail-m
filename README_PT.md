## Oque é isso?
Coisas úteis para rodar Linux em dispositivos baseados em Bay Trail-m: Dell Venue 8 Pro, Lenovo Miix 2, Acer Iconia W4, Asus T100, Toshiba Encore...

## Oque há de Util?
* alsa state file.
* firmwares.
* xorg config para o display.
* e mais.
